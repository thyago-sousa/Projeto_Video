# Projeto AulaAndroid Tube

Aplicativo reprodutor de mídia desenvolvido em **Android Studio (Java)** para execução de vídeos locais no dispositivo.

##  Aluno: Thyago

---

## Descrição

O sistema gerencia uma lista de reprodução permitindo que o usuário assista aos vídeos com controle completo de navegação, reprodução e acompanhamento em tempo real.

O aplicativo utiliza componentes nativos do Android como `VideoView`, `SeekBar`, `Handler` e controles personalizados para proporcionar uma experiência simples e funcional de reprodução de mídia.
